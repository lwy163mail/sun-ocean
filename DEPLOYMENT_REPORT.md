## 🚀 访问指南

### 1. 打开浏览器
在浏览器地址栏输入：`http://localhost:3000`

### 2. 导航操作
- **点击分类卡片** → 查看该分类下的所有链接
- **点击链接卡片** → 在新标签页打开链接
- **点击"管理"菜单** → 进入管理页面
- **点击"添加链接"** → 添加新的收藏链接
- **点击"添加分类"** → 创建新的分类

### 3. 功能体验
1. **分类浏览** - 查看不同分类的链接
2. **链接访问** - 点击链接直接访问
3. **访问统计** - 查看每个链接的访问次数
4. **内容管理** - 添加/编辑/删除链接和分类
5. **数据视图** - 查看JSON格式的数据
